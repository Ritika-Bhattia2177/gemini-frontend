Frequently Asked Questions

Everything you need to know

Where is my API key used?

All API keys are stored securely on the server backend. The frontend never has access to keys, and they are never sent to the browser for security.

Why demo responses?

If no API key is configured, the app returns safe demo answers so you can test the UI and experience the interface before connecting to the real AI.

How fast are the responses?

Responses typically arrive in 2-5 seconds, powered by Google's Gemini AI. Speed may vary based on question complexity and API response time.

Is my data private?

Yes! Your conversations are sent directly to Google's Gemini API using your own API key. We don't store or log any of your questions or responses.

What can I ask Gemini?

Anything! Ask for explanations, code help, creative writing, summaries, translations, math solutions, or general knowledge questions. Gemini is versatile and knowledgeable.
